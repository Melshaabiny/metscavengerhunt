class user_auth.models.QuestionAsked(*args, **kwargs)[source]¶

This model QuestionAsked is related to the questions that a user asked on the website. It has title, description and the status which indicate if the question is answered or not. This model should be one-to-many from user to QuestionAsked since each user can have many asked questions.
